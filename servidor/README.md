Nmap-Web-Client (Server)
============

#### Requirements
* [node-pcap](https://github.com/mranney/node_pcap) by mranney
* libpcap-dev (For debian you can use **apt-get install libpcap-dev**)

### Installation

```
  $ npm install
```

### How to run the server

** You must be root for running.

```
  $ sudo node server.js
```
